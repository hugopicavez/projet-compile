TP Langages et Compilation > Introduction


Ce projet a pour buts :
  • écrire un compilateur "zéro-défault" pour le langage JCas ;
  • utiliser des générateurs d'analyseurs lexical et syntaxique (JFlex et Cup) ;
  • écrire des tests ;
  • travailler en équipe.

Transparents du cours


Instruction pour la récupération et l'installation du projet

Installation sur machine personnelle (Système Linux)

  • Java doit être installé (version 1.5 minimum)
  • Pour faire fonctionner la machine abstraite (pour la passe 3), il sera nécessaire d'installer le compilateur Ada Gnat.

Téléchargement et installation du projet

  • Télécharger le projet : ProjetCompil.tar.gz
  • Placer le fichier ProjetCompil.tar.gz dans le répertoire $HOME. Détarer et décompresser :
    gunzip ProjetCompil.tar.gz
    tar xvf ProjetCompil.tar

Modifier le CLASSPATH

  • Il faut rajouter jflex et CUP au CLASSPATH pour pouvoir compiler le projet fourni.
  • Dans le fichier .bash_profile , (ou .bashrc , ou .tcshrc...), modifier le CLASSPATH : 
    export CLASSPATH=$CLASSPATH:$HOME:$HOME/ProjetCompil/Global/Bin/java-cup-11a-runtime.jar:$HOME/ProjetCompil/Global/Bin/JFlex.jar:.
    Dans le dossier $HOME, relire le fichier .bash_profile : source .bash_profile, (ou source .bashrc , ou .tcshrc...)

Compiler

  • cd ProjetCompil
  • make

Tester l'analyse lexicale

  • cd Syntaxe/Test
  • ./lexico simple.cas
