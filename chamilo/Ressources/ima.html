Installation de la machine abstraite

Pour exécuter les programmes assembleurs générés, il est nécessaire d'installer l'interprète de machine abstraite (ima).

Installation de Gnat

L'interprète de machine abstraite est écrit en Ada, il faut donc installer le compilateur Ada Gnat (commande gnatmake).

Sous Ubuntu :
sudo apt-get install gnat

Sous Debian :
aptitude install gnat

Sous Fedora :
yum install gcc-gnat

Installation de l'interprète de machine abstraite (ima)

Télécharger les sources de la machine abstraite
gunzip Interp_MA.tar.gz
tar xvf Interp_MA.tar
cd Interp_MA/Src
make

L'exécutable ima se trouve dans Interp_MA/Exec

cp ../Exec/ima ~/ProjetCompil/Bin

Tester la machine abstraite

Créer le fichier essai.ass contenant les lignes

WSTR "ok"
WNL
HALT

Effectuer la commande ima essai.ass
Cette commande doit afficher "ok".
